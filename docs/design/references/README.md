# SchoolOS Design References

**Status:** Reference index  
**Last updated:** 2026-06-15

This folder is reserved for long-form design reference documents.

Reference documents explain reasoning, detailed alternatives, screen ideas, and deep background. They are useful, but they are not the daily source of truth.

## What Belongs Here

```text
Detailed role/screen design explanations
Deep UI/UX reasoning
Uploaded plan consolidation notes
Alternative screen patterns
Long-form component and module design references
```

---

## What Does Not Belong Here

```text
Active implementation checklists
Screen contract matrices
Permission catalogs
Backend contract rules
Persona smoke tests
Old/superseded drafts
```

Implementation-facing frontend docs belong under `docs/frontend/`.
Old or superseded docs belong under `docs/archive/`.
