# SchoolOS

SchoolOS is a production-grade, Nepal-first multi-tenant education operating SaaS.

Active product scope:

```text
PRESCHOOL
  Montessori, Nursery, LKG, UKG

SCHOOL
  Grade 1-10

HIGHER_SECONDARY
  Grade 11-12 / +2

BACHELOR
  Bachelor's-level institutions/programs
```

Master's is not an active full institution-management pack. It is a future higher-education extension and an allowed Student App eligibility level only.

Current readiness remains documented in `docs/project/SCHOOLOS_PRODUCTION_READINESS_AUDIT.md`.
