# SchoolOS

SchoolOS is a multi-tenant SaaS platform for Nepal education institutions.

Supported experience packs:

```text
SCHOOL
  Grade 1-10

HIGHER_SECONDARY
  Grade 11-12 / +2

COLLEGE
  Bachelor, Master, Diploma, and professional programs
```

The product now focuses on Grade 1-10, Grade 11-12, and College. Earlier preschool scope has been removed from active documentation.
